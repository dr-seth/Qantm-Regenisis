# Qantm-Regenisis

QantmOrchstrtr verticals for trading and regenesis (distressed real estate to data centers + senior living).

## Verticals

### Trading
Algorithmic trading vertical with nano models for price prediction, strategy optimization, and risk management.

### ReGenesis
Convert distressed, closed, or underperforming enclosed real estate into resilient, multi-program Senior Living Campuses and Data Centers.

**37 Nano Models** including:
- Core models: site infrastructure, autonomous intelligence, compliance, energy, etc.
- Data center bespoke models: cooling, power, solar, geothermal, latency evaluation

## Setup

```bash
# Install dependencies
pip install -r requirements.txt

# Run tests
pytest verticals/
```
